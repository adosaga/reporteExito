Para ejecutar el reporte:

1. Descargar los archivos reporteExito.jar y config.json.
2. Configurar el archivo config.json con las credenciales, rutas donde se escribirá los reportes, etc.
3. Los ejecutables de java(.jar) se ejecutan vía cónsola, para este reporte se debe ejecutar lo siguiente:

        >> java -jar reporteExito.jar

4. Si la máquina no tiene instalada la JVM de java, descargarla e instalarla desde https://www.java.com/es/download/

NOTA: Los archivos resultados se alojarán en la ruta localpath especificada en el archivo config.json
el config.json debe ubicarse en el mismo directorio que el ejecutable .jar
